ShopSphere – Full Stack E-Commerce Website

Developed a e-commerce platform using Django integrated with authentication (login, register, profile,
logout, reset/forgot password) and product/cart management.
Implemented product catlog with search, category filters, trending and offer sections, and integrated REST
API for dynamic cart and product updates.
Built shopping cart functionality including add, remove, increment, decrement operations, and real-time
total price calculation using Django ORM.
Applied user authentication, session management, and secure password handling with Django’s built-in
authentication system to ensure a secure user experience
